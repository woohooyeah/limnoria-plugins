If a Spotify URI is posted, this plugin replies with Artist and Song name. Can reply with HTTP URL aswell. 
May also use the Spotify API to reply with Artist and Song name on HTTP URLs from open.spotify.com

Examples:
12:34:42 <@Hoaas> spotify:track:3WoF0QidrggYV482mOb7ye
12:34:44 <@Bunisher> >> Bush - Glycerine (http://open.spotify.com/track/3WoF0QidrggYV482mOb7ye)

12:35:28 <@Hoaas> http://open.spotify.com/track/3WoF0QidrggYV482mOb7ye
12:35:28 <@Bunisher> >> Bush - Glycerine

12:36:18 <@Hoaas> spotify:artist:78SHxLdtysAXgywQ4vE0Oa
12:36:20 <@Bunisher> >> Bush (http://open.spotify.com/artist/78SHxLdtysAXgywQ4vE0Oa)

12:36:42 <@Hoaas> spotify:user:drminty:playlist:5k68VkdqfTU5akVpYpdD6D
12:36:44 <@Bunisher> http://open.spotify.com/user/drminty/playlist/5k68VkdqfTU5akVpYpdD6D

12:37:16 <@Hoaas> spotify:user:test
12:37:18 <@Bunisher> http://open.spotify.com/user/test
